Keyboard Shortcuts

You have a number of keyboard shortcuts available to you:

  • ?: Access and close this dialog
  • 1: Go to first slide
  • 2-9: Go to around that point (percentage-wise) of the presentation
  • 0: Go to last slide
  • . (period): Dim presentation (press again to bring it back)
  • Esc: Restore to presentation mode (remove this dialog, undim presentation)
  • X: Export generated templates to console (useful if you want to save transitions)
You can also have custom keyboard shortcuts for slide groups.
Simply add a 'key' property to the group with the shortcut key.
